# Twitter Bootstrap 3.2.1 for Rails 4 Asset Pipeline
Bootstrap is a toolkit from Twitter designed to kickstart development of webapps and sites. It includes base CSS and HTML for typography, forms, buttons, tables, grids, navigation, and more.

twitter-bootstrap-rails project integrates Bootstrap CSS toolkit for Rails Asset Pipeline (Rails 4, 3.1, 3.2 are supported)

## Quick install

You can install the gem using [LESS](https://github.com/seyhunak/twitter-bootstrap-rails/wiki/Installing-using-LESS) or plain [CSS](https://github.com/seyhunak/twitter-bootstrap-rails/wiki/Installing-using-plain-CSS)

Most of the info you'll need is in the wiki, and if it isn't please contribute!
